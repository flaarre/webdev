### Webdev

# Phase 1 — HTML & CSS Basics
**Duration:** 2 Weeks | **Time:** 20 min/day

---

## Days 1–3: HTML Structure

### What You'll Learn
- What HTML is and how the browser reads it
- How to create a valid HTML document from scratch (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Core tags and when to use each:
  - `<h1>` through `<h6>` — headings
  - `<p>` — paragraphs
  - `<div>` — generic container/block element
  - `<span>` — inline container
  - `<a href="">` — hyperlinks
  - `<img src="">` — images
  - `<ul>`, `<ol>`, `<li>` — lists
  - `<br>` — line break

### What You'll Be Able to Build
- A plain HTML page with a heading, paragraph, image, and a working link
- Understand the difference between block vs inline elements

---

## Days 4–7: CSS Basics

### What You'll Learn
- How to link a CSS file to your HTML (`<link rel="stylesheet">`)
- CSS selectors:
  - Element selector — `h1 { }`
  - Class selector — `.card { }`
  - ID selector — `#title { }`
- Core properties:
  - `color`, `background-color`
  - `font-family`, `font-size`, `font-weight`
  - `margin`, `padding`, `border`
  - `width`, `height`
- The box model — every element is a box with content, padding, border, and margin

### What You'll Be Able to Build
- Style your HTML page with custom fonts, colors, and spacing
- Understand why elements shift when you change margin vs padding

---

## Days 8–10: CSS Layout — Flexbox

### What You'll Learn
- Why normal HTML stacks elements vertically by default
- `display: flex` — turns a container into a flex layout
- Key flex properties:
  - `flex-direction` — row or column
  - `justify-content` — horizontal alignment (center, space-between, etc.)
  - `align-items` — vertical alignment
  - `gap` — spacing between items
  - `flex-wrap` — wrapping behavior

### What You'll Be Able to Build
- A centered layout
- A horizontal navigation bar
- A row of cards side by side

> Flexbox is the most important CSS concept. Don't rush this — spend real time here.

---

## Days 11–14: Capstone — Static Personal Page

### What You'll Build
A fully styled personal page with no JavaScript. Must include:

- [ ] Your name as an `<h1>` heading
- [ ] A short bio paragraph
- [ ] A profile image
- [ ] A navigation bar with links (built with flexbox)
- [ ] At least 2 sections with different background colors
- [ ] Clean, readable CSS in a separate `styles.css` file

### Rules
- No copy-pasting from tutorials — write every line yourself
- If you get stuck, check [MDN Web Docs](https://developer.mozilla.org) only
- Must look intentional — not just black text on white background
